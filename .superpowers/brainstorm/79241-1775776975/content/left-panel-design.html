Left Panel: Vector Controls + Model Info

What should the left column contain and how should it be organized?

A

Stacked Sections: Model → Vectors → Controls → Config

Four distinct sections, top to bottom. Model info is static at top, vectors list is scrollable, controls show the selected vector's parameters, config shows generation settings.

Compact: Model Header + Unified Vector/Control List

Model info compressed into 1-2 lines at top. Vectors and their controls are integrated — each vector row expands inline when selected to show alpha bar and layer. Generation config at bottom.
